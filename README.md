# MineTS - Mine This Shit

## ⚠️ Project Status

> Development on this project has been paused until I *get good* and learn about linear algebra and related topics.

What started as a project to learn coding while being guided by AI and have fun (Minecraft-style project in Three.js) quickly turned into something *way, **WAY*** beyond my current level as a programmer.

It ended up touching areas like rendering pipelines, meshing, collision detection, and spatial math. Coming from web development, it got to the point where I couldn’t even comprehend what the hell the AI was explaining or asking me to do.

The short version: 3D, voxels, and game engine programming are hard. Like, really hard.

No wonder people take years to build this kind of stuff.
